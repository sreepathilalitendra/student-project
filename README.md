#Student Record Management System

This is a simple Python-based project that allows users to manage student records.

Features:
- Add student details (Name, Roll Number, Marks)
- Display all student records
- Delete a student record using roll number
- Menu-driven interface for easy interaction

Concepts Used:
- Python lists
- Dictionaries
- Functions
- Loops
- Conditional statements

This project is created for learning and practicing basic Python programming concepts.
